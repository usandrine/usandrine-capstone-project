# Expense Tracker

A full-featured web application for tracking personal income and expenses, built with Django.

## Features

- **User Authentication**: Secure registration and login system
- **Dashboard**: Overview of financial status with total income, expense, and balance
- **Transaction Management**: Add, edit, delete, and filter transactions
- **Category System**: Organize expenses by custom categories
- **Monthly Overview**: Automatic calculation of monthly totals
- **Responsive Design**: Works on both desktop and mobile devices
- **Filtering**: Filter transactions by category, type, and date range

## Tech Stack

- **Backend**: Django, Python
- **Frontend**: HTML, CSS, Bootstrap 5
- **Database**: SQLite (default), can be configured for PostgreSQL/MySQL
- **Authentication**: Django's built-in authentication system
